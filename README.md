# ENOSE Sensor Data Visualization
eNose Sensor - ARTIK 530s - Cloud Demo Project

## Requirements
* eNose sensor
* Nodejs
* Accounts for ARTIK Cloud, Azure
* Web app for Azure (https://github.com/sskcorea/enose-webapp)

## Start
```
npm install
node index
```
